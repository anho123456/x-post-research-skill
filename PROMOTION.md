## 小红书 / 公众号风格

标题候选：

- 我把“让 AI 读懂 X 帖子”做成了一个开源 Skill
- 新手也能学：从一个真实问题做出 Agent Skill
- 免费开源：让 Codex / Claude / Hermes 总结 X/Twitter 帖子

正文：

最近我经常把 X/Twitter 上的教程帖、工具帖发给 AI，让它帮我总结。但只让 AI “总结一下”经常不够，它可能只看表面文字，忽略图片、引用帖、评论区和外链。

所以我做了一个通用 Agent Skill：x-post-research-skill。

它的目标不是“完美读取所有 X 内容”，而是给智能体一个更可靠的工作流程：

1. 先尝试公开接口读取帖子。
2. 失败时尝试用本机 Chrome 登录态读取。
3. 把正文、图片、引用、评论和外链分开核查。
4. 用中文输出主要内容、可实践步骤、风险和待核查点。

这个项目适合两类人：

- 想让 AI 更靠谱地读 X/Twitter 帖子的人。
- 想学习如何把真实问题沉淀成 Agent Skill 的新手。
